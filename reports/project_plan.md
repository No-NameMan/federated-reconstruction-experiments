# Краткий план проекта

Основная цель: исследовать Federated Reconstruction для matrix factorization на MovieLens 1M в условиях ограниченной коммуникации.

Фокус экспериментов:

1. Базовый FEDRECON.
2. Ablation по reconstruction steps.
3. Ablation по client update steps.
4. Server optimizers.
5. Quantization of global deltas.
6. Communication-quality trade-off.
7. Client sparsity.
8. Dropout simulation.

Основной результат: PyTorch-симулятор + графики для курсовой и презентации.
